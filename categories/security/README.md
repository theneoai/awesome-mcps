# 安全 MCP

> 密钥管理、漏洞扫描、代码安全审计等安全相关 MCP 服务器。

## 密钥与凭证管理

| 名称 | 描述 | 语言 | 链接 |
|------|------|------|------|
| 1Password MCP | 1Password 密钥与凭证安全读取（官方） | Go | [github.com/1Password/mcp-server](https://github.com/1Password/mcp-server) |
| HashiCorp Vault MCP | Vault 动态密钥与 Secret 管理 | Go | [github.com/gmllt/mcp-vault](https://github.com/gmllt/mcp-vault) |

## 代码安全扫描

| 名称 | 描述 | 语言 | 链接 |
|------|------|------|------|
| Semgrep MCP | Semgrep 静态代码安全分析（官方） | Python | [github.com/semgrep/mcp](https://github.com/semgrep/mcp) |
| Snyk MCP | Snyk 开源依赖漏洞与许可证检测 | TypeScript | [github.com/snyk/snyk-mcp](https://github.com/snyk/snyk-mcp) |
| Bandit MCP | Python 代码安全扫描（基于 Bandit） | Python | 社区维护 |

> **注意**：安全类 MCP 服务器在使用时应严格遵循最小权限原则，仅授予必要的访问权限。
