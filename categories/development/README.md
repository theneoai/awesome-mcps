# 开发工具 MCP

> 面向软件开发的 MCP 服务器，涵盖代码托管、CI/CD、容器、浏览器自动化等场景。

## 代码托管

| 名称 | 描述 | 语言 | 链接 |
|------|------|------|------|
| GitHub MCP | GitHub 官方 MCP，支持仓库、PR、Issue、Actions 完整操作 | Go | [github.com/github/github-mcp-server](https://github.com/github/github-mcp-server) |
| GitLab MCP | GitLab 平台集成，支持 MR、流水线、Wiki | TypeScript | [github.com/zereight/gitlab-mcp](https://github.com/zereight/gitlab-mcp) |
| Bitbucket MCP | Bitbucket Cloud 仓库与 PR 管理 | Python | [github.com/aditya-bhuyan/mcp-bitbucket](https://github.com/aditya-bhuyan/mcp-bitbucket) |

## 容器与编排

| 名称 | 描述 | 语言 | 链接 |
|------|------|------|------|
| Docker MCP | 管理容器、镜像、网络、卷 | Python | [github.com/ckreiling/mcp-server-docker](https://github.com/ckreiling/mcp-server-docker) |
| Kubernetes MCP | 集群资源管理，支持 Pod、Deployment、Service | TypeScript | [github.com/Flux159/mcp-server-kubernetes](https://github.com/Flux159/mcp-server-kubernetes) |

## 浏览器自动化

| 名称 | 描述 | 语言 | 链接 |
|------|------|------|------|
| Playwright MCP | 微软官方浏览器自动化，支持截图、点击、表单填写 | TypeScript | [github.com/microsoft/playwright-mcp](https://github.com/microsoft/playwright-mcp) |
| Puppeteer MCP | Puppeteer 浏览器控制（官方参考实现） | TypeScript | [github.com/modelcontextprotocol/servers](https://github.com/modelcontextprotocol/servers/tree/main/src/puppeteer) |

## 文件系统

| 名称 | 描述 | 语言 | 链接 |
|------|------|------|------|
| Filesystem MCP | 本地文件系统读写，支持目录树与搜索（官方） | TypeScript | [github.com/modelcontextprotocol/servers](https://github.com/modelcontextprotocol/servers/tree/main/src/filesystem) |
| Git MCP | Git 仓库操作，commit、diff、log、branch（官方） | Python | [github.com/modelcontextprotocol/servers](https://github.com/modelcontextprotocol/servers/tree/main/src/git) |
