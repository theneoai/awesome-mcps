# AI 与机器学习 MCP

> 连接 AI 模型、推理平台与机器学习框架的 MCP 服务器。

## 模型推理

| 名称 | 描述 | 语言 | 链接 |
|------|------|------|------|
| Hugging Face MCP | Hugging Face Spaces 模型推理与 API | TypeScript | [github.com/evalstate/mcp-hfspace](https://github.com/evalstate/mcp-hfspace) |
| Replicate MCP | Replicate 平台图像、音频、视频模型调用 | TypeScript | [github.com/deepfates/mcp-replicate](https://github.com/deepfates/mcp-replicate) |

## AI 框架集成

| 名称 | 描述 | 语言 | 链接 |
|------|------|------|------|
| LangChain MCP Adapters | LangChain 工具与 MCP 双向适配 | Python | [github.com/langchain-ai/langchain-mcp-adapters](https://github.com/langchain-ai/langchain-mcp-adapters) |
| OpenAI Agents MCP | OpenAI Agents SDK 内置 MCP 客户端支持 | Python | [github.com/openai/openai-agents-python](https://github.com/openai/openai-agents-python) |
